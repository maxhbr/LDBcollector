The License Database

To access the database use the Linked Data Fragments server at licensedb.org/data.

You can also download full database dumps from licensedb.org/dl/.

Adding or correcting data in the database

If you want to add an entry to the database, or edit an existing entry in the database, please fork or clone the repository on github. The file EDITING.txt in the project root contains instructions and guidelines for adding data to the database.
