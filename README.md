Build
=====

Make sure GNU make, python 2.7 and a recent version of io.js / node.js is installed, then
run:

    bin/bootstrap.sh
    make

To generate the HDT dump RDFHDT needs to be installed, download that from
http://www.rdfhdt.org/download/
